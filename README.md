# H5 原型使用说明

## 打开方式

直接双击打开：

```text
index.html
```

或在浏览器中打开本文件夹下的 `index.html`。

## 当前能力

- 选择结果卡形象：男生版 / 女生版。
- 完成 20 道正式测试题。
- 全部题目为单选题。
- 自动计算九型得分、主型、侧翼和混合倾向。
- 展示完整结果页文案。
- 根据性别形象调用对应分享卡。
- 支持复制朋友圈文案。

## 文件结构

```text
H5原型/
├── index.html
├── styles.css
├── questions-data.js
├── app.js
└── README.md
```

素材来自上级目录：

```text
./assets/characters/
./assets/share_cards/
```
