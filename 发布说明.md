# 朋友圈 H5 发布说明

## 1. 这个文件夹是什么

这是《测测你的工位精神体：公司越疯，你越像谁？》的朋友圈 H5 发布包。

上传整个文件夹到一个 HTTPS 服务器后，访问其中的 `index.html` 就可以开始测试并转发。

## 2. 必须上传的内容

请保持以下结构不变：

```text
朋友圈H5发布包/
├── index.html
├── styles.css
├── app.js
├── questions-data.js
├── share-config.js
└── assets/
```

不要只上传 `index.html`，否则结果卡、角色图和分享封面会丢失。

## 3. 推荐发布方式

可以上传到任意支持 HTTPS 的静态托管：

- 腾讯云 COS 静态网站
- 阿里云 OSS 静态网站
- 七牛云
- Vercel
- Netlify
- 公司官网服务器

发布后访问地址应类似：

```text
https://你的域名/人格测试/index.html
```

## 4. 朋友圈转发说明

普通 H5 页面可以被用户从微信右上角转发。

页面内已加入：

- 分享标题
- 分享描述
- 分享封面图
- 微信内右上角转发引导
- 普通浏览器系统分享/复制链接能力

注意：微信朋友圈想稳定显示自定义标题、描述和封面，通常需要接入微信 JS-SDK，并由服务端生成签名。这个静态包已经准备好前端分享信息，但不包含微信签名服务。

## 5. 分享配置

基础分享信息在：

```text
share-config.js
```

可修改：

```js
window.H5_SHARE_CONFIG = {
  title: "测测你的工位精神体：公司越疯，你越像谁？",
  desc: "公司副本一开，精神值就开始漏电。20 题检测你的工位精神体会觉醒成什么形态。",
  image: "./assets/share_cover.png",
  assetBase: "./assets"
};
```

如果未来接入微信 JS-SDK，建议继续沿用这里的 title、desc、image。

## 6. 测试清单

上线前请检查：

- 首页能打开。
- 能选择男生/女生形象。
- 20 道题能完整答完。
- 结果页能正常显示对应分享卡。
- 点击“转发这个测试”在微信内能出现右上角引导。
- 分享封面图 `assets/share_cover.png` 能通过线上 URL 直接打开。
